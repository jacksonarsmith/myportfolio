## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jacksonarsmith/myportfolio.git
   cd myportfolio

2. **Install dependencies**:
    ```
    npm install

3. **Run application in dev with vite**:
    ```
    npm run dev
